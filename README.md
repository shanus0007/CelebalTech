# Data Cleaning and Analysis using Pandas

## Internship Assignment Overview
This assignment was completed as part of my internship to demonstrate practical knowledge of data preprocessing and analysis using Python and Pandas. The objective was to clean, explore, and transform a CSV dataset for better analysis and usability.

## Tasks Performed
- Loaded the CSV dataset into a Pandas DataFrame
- Explored the dataset using:
  - `head()`
  - `tail()`
  - `shape`
  - `columns`
  - `dtypes`
- Identified and handled missing values using:
  - `fillna()`
  - `dropna()`
- Removed duplicate records using `drop_duplicates()`
- Performed basic data operations:
  - Filtering rows
  - Selecting columns
- Created a derived column:
  - `total_amount`
- Exported the cleaned dataset as a new CSV file

## Technologies Used
- Python
- Pandas
- Jupyter Notebook

## Output
A cleaned and processed dataset was successfully generated and saved as a new CSV file for further analysis and reporting.

## Learning Outcome
Through this assignment, I gained hands-on experience in:
- Data cleaning
- Handling missing values
- Data transformation
- Working with Pandas DataFrames
- Exporting processed datasets

## Conclusion
This internship assignment helped strengthen my understanding of essential data preprocessing techniques used in real-world data analysis workflows.
